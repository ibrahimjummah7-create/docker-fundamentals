This container serves a very small static website using nginx. It demonstrates how to package static content into a container image so the content can be run the same way across environments. In a larger system this image could be deployed behind a load balancer, served by an orchestration platform (Kubernetes, Docker Swarm) or used as a static asset origin for a microservice architecture.
